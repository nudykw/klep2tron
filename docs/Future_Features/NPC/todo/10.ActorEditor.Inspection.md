# 10. Режим инспекции (Post-Slicing)

> [!IMPORTANT]
> Данный этап выполняется ПОСЛЕ реализации разрезания модели (задача 04).

## 1. Визуализация плоскостей
- [ ] Реализовать отрисовку плоскостей разреза как полупрозрачных мешей.
- [ ] Динамическое обновление положения плоскостей при изменении параметров `cut_top` / `cut_bottom`.

## 2. Инспекция частей (Head/Body/Engine)
- [ ] При активации режима инспекции конкретной части:
    - [ ] Скрывать остальные части модели.
    - [ ] Центрировать камеру на Bounding Box выбранной части.
    - [ ] Ограничивать вращение камеры вокруг центра этой части.
- [ ] Реализовать кнопку "Focus" в инспекторе для каждой части.
