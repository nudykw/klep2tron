# 02. Импорт модели

> [!IMPORTANT]
> Работа ведется исключительно в ветке **feat/actor-editor**.

## 1. Выбор файла
- [ ] Интегрировать диалоговое окно выбора файла (например, через `rfd` или встроенный в `egui` файловый менеджер).
- [ ] Ограничить выбор форматами: `.gltf`, `.glb`, `.obj`.

## 2. Загрузка в Bevy
- [ ] Использовать `AssetServer` для асинхронной загрузки модели.
- [ ] Реализовать "Loading Screen" или индикатор загрузки.
- [ ] После загрузки:
    - [ ] Вычислить Bounding Box модели.
    - [ ] Автоматически отмасштабировать и центрировать модель во вьюпорте.
    - [ ] Извлечь исходную геометрию (Mesh) для хранения в `OriginalMeshComponent`.
