# Задача 14.2: Инфраструктура Manual режима (Base)

> [!IMPORTANT]
> Работа ведется исключительно в ветке **feat/actor-editor**.

> [!NOTE]
> Общее описание системы ручной нарезки находится в [Задаче 14](./14.Manual_Mesh_Slicing.md). В данной подзадаче необходимо реализовать только функционал, описанный ниже.
Реализовать логику "заморозки" меша и переключения состояний системы нарезки.

## Шаги
1.  **State Management:** Реализовать логику переключения ресурса `SlicingSettings`.
2.  **Заморозка (Freezing):** При активации `Manual Mode` прекращать процедурный вызов `split_mesh_by_planes` и фиксировать текущие 3 меша (Head, Body, Legs).
3.  **Скрытие слайдеров:** В ручном режиме полностью скрывать визуальные плоскости нарезки из вьюпорта.
4.  **Сохранение и Загрузка:** 
    - Добавить маркер режима в метаданные проекта.
    - При загрузке проекта в `Manual Mode` автоматически отключать слайсеры и переходить в статический режим отображения.

## Ожидаемый результат
При нажатии кнопки Manual меш перестает реагировать на слайдеры (которые исчезают), а при перезагрузке проекта состояние Manual сохраняется.
