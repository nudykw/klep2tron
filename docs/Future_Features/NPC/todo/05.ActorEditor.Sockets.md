# 05. Сокеты

> [!IMPORTANT]
> Работа ведется исключительно в ветке **feat/actor-editor**.

## 1. Установка и Манипуляция
- [ ] Реализовать Picking (Raycast) для определения точки на поверхности.
- [ ] При клике спавнить `SocketEntity` с дефолтным именем.
- [ ] Интегрировать 3D-Gizmo (например, `bevy_transform_gizmo`) для перемещения и вращения сокета.

## 2. Метаданные и Валидация
- [ ] Реализовать UI-форму для редактирования Имени и Комментария.
- [ ] **Валидация имен**:
    - [ ] При попытке дать имя, которое уже есть в текущей группе (например, две "Eye_L" в голове) — выдавать ошибку или добавлять индекс.
- [ ] Автоматическая привязка сокета к ближайшей части (Head/Body/Engine) при создании.

## 3. Визуализация
- [ ] Отрисовка сокетов в виде 3D-иконок или стрелок, показывающих направление (Forward).
- [ ] Цветовая кодировка сокетов по группам.
- [ ] Список всех сокетов в интерфейсе.

## 4. Система спецэффектов (VFX)
- [ ] Определить структуру `EffectConfig` в `shared` для описания параметров (тип, цвет, скорость).
- [ ] Реализовать загрузку пресетов эффектов из внешних файлов (например, `.ron`).
- [ ] Создать базовый визуализатор (эмиттер частиц или шейдерный эффект) для превью в сокете.
- [ ] Добавить в UI редактора возможность выбора и настройки эффекта для каждого сокета.
- [ ] Реализовать опцию «Show/Hide Previews» в глобальных настройках визуализации.
