# 05. Сокеты

> [!IMPORTANT]
> Работа ведется исключительно в ветке **feat/actor-editor**.

## 1. Манипуляция и Визуализация (MVP)

- [x] **Режим установки**: Ray-cast клик по мешу для спавна сокета.
- [x] **3D Translation**: Перемещение по осям X, Y, Z (World aligned).
- [x] **3D Rotation**:
  - [x] Гизмо-кольца (Red-X, Green-Y, Blue-Z).
  - [x] Стабильный расчет (без прыжков и накопления ошибок).
  - [x] Кнопка сброса (Reset Rotation) в инспекторе.
- [x] **Индикатор Forward**:
  - [x] Визуализация через Pin с наконечником-конусом.
  - [x] Направление по локальной оси Y (нормаль меша).
- [x] **UX Полировка**:
  - [x] Приоритет выбора осей (одна за раз).
  - [x] Блокировка других элементов во время активного Drag.
  - [x] Авто-привязка к ближайшей части (ECS Hierarchy).

## 2. Метаданные и Валидация

- [x] **Редактирование данных**:

## 5. Библиотека пресетов (Базовый набор)

- [ ] **Plasma Flame (Пламя плазмы)**:
  - [ ] Реализовать градиент Cyan -> Deep Blue -> Purple.
  - [ ] Настроить эффект пульсации масштаба (Pulse logic).
  - [ ] Добавить легкий Bloom-эффект в превью.
- [ ] **Muzzle Flash (Вспышка)**:
  - [ ] Реализовать мгновенный Burst-спавн (время жизни < 0.1s).
  - [ ] Форма: "Звезда" или перекрещенные плоскости.
  - [ ] Цвет: Bright Yellow -> Orange.
- [ ] **Smoke (Дым)**:
  - [ ] Реализовать медленное движение вверх (Force/Velocity).
  - [ ] Плавное исчезновение (Alpha Fade-out).
  - [ ] Увеличение размера со временем (Scale over lifetime).
- [ ] **Интеграция**:
  - [ ] Кнопка "Apply Preset" в инспекторе.
  - [ ] Возможность быстрого переключения между типами для сравнения.
