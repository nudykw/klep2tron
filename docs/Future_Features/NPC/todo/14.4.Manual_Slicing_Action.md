# Задача 14.4: Переназначение меша (Geometry Reassignment)

> [!IMPORTANT]
> Работа ведется исключительно в ветке **feat/actor-editor**.

> [!NOTE]
> Общее описание системы ручной нарезки находится в [Задаче 14](./14.Manual_Mesh_Slicing.md). В данной подзадаче необходимо реализовать только функционал, описанный ниже.
Реализовать физический перенос геометрии между частями и логику очистки крышек.

## Шаги
1.  **Target Part Selector:** Реализовать комбобокс в правой панели (Head / Body / Legs).
2.  **Keep Caps Logic:** Реализовать чекбокс "Keep Caps" (Оставить крышки) с тултипом: "Сохраняет заглушки на срезе. Отключите, чтобы избежать внутренних перегородок при объединении разрезанных частей."
    - Если чекбокс выключен при переходе в Manual — удалять процедурные треугольники крышек.
3.  **Geometry Transfer:** Реализовать логику переноса треугольников (вершин и индексов) из меша одной части в меш другой при смене значения в комбобоксе.
4.  **Undo/Redo:** Регистрация каждого факта переноса в системе Undo/Redo.

## Ожидаемый результат
Пользователь может выбрать область лассо и через комбобокс перенести её в другую часть тела. Меши обновляются мгновенно, действие можно отменить.
