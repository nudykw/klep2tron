# 01. Создание интерфейса

## 1. Пункт меню

- [ ] Добавить в `MainMenuState` новый пункт "Actor Editor".
- [ ] Реализовать переход в состояние `GameState::ActorEditor`.
- [ ] При переходе выгружать игровые ассеты и загружать окружение редактора (чистая сцена, освещение).
- [ ] Реализовать отображение пустого окна редактора (базовый UI-узел) при входе в состояние.

## 2. Окно с контролами (Sidebar)

- [ ] Реализовать левую панель (Project Management):
  - [ ] Кнопки: Import, Open, Save.
  - [ ] Переключатель режимов (Slicing/Sockets).
- [ ] Реализовать правую панель (Inspector):
  - [ ] Виджет Range Slider (вертикальный).
  - [ ] Секция материалов:
    - [ ] Color Picker.
    - [ ] Цветовые пресеты (палитра для быстрой смены цветов).
  - [ ] Список сокетов:
    - [ ] Поле поиска (Search Bar).
    - [ ] Группировка и список.

## 3. Инструментарий и навигация (Viewport UI)

- [ ] Реализовать панель инструментов вьюера (Top Toolbar):
  - [ ] Переключатели видимости: Grid, Slices, Sockets, Gizmos.
  - [ ] Кнопка сброса камеры (Reset View).
- [ ] Реализовать виджет навигации (Navigation Gizmo):
  - [ ] 3D-оси в углу экрана для отображения ориентации.

## 4. Статус-бар и уведомления

- [ ] Реализовать нижнюю панель (Status Bar):
  - [ ] Отображение текущего состояния (Ready/Saving/Processing).
  - [ ] Статистика модели (Polycount).
- [ ] Реализовать систему оверлеев:
  - [ ] Toast-уведомления (Save success, Import error).
  - [ ] Модальное окно подтверждения (Discard changes).

## 5. Макет и адаптивность (Layout)

- [ ] Реализовать изменение ширины панелей (Resizable):
  - [ ] Добавить интерактивную границу (Handle) между панелями и центральной областью.
  - [ ] Использовать `RelativeCursorPosition` и `Style::width` для динамического изменения размера.
  - [ ] Установить ограничения (например, min: 200px, max: 40% ширины окна).
  - [ ] Визуальный отклик: менять курсор на `ResizeLeftRight` при наведении на границу.
- [ ] Адаптивность к размеру окна:
  - [ ] Реализовать систему синхронизации `Camera::viewport` с размерами центрального UI-узла.
  - [ ] Центральная область с 3D-вьюером должна автоматически пересчитывать размер при изменении ширины панелей или окна.
  - [ ] Элементы внутри панелей должны корректно перестраиваться или обрезаться при уменьшении ширины.

## 6. Ограничения

- [ ] Платформенные ограничения: Редактор не поддерживается в Web (WASM).
  - [ ] Использовать `#[cfg(not(target_arch = "wasm32"))]` для исключения логики редактора из веб-сборки.
  - [ ] Пункт меню должен быть скрыт при запуске в браузере.
  - [ ] Добавить соответствующее уведомление, если пользователь пытается открыть редактор в вебе.
