# 04. Actor Editor: Slicing & Capping Implementation Plan

## Описание задачи
Разделение монолитной импортированной модели на функциональные части (Голова, Туловище, Ноги) путем разрезания меша по горизонтальным плоскостям. Это необходимо для обеспечения модульности и независимого управления частями (например, вращение головы робота).

## Почему мы это делаем?
1. **Модульность**: Возможность менять части NPC (разные головы на одном теле).
2. **Анимация**: Независимый контроль частей (вращение головы, анимация ног).
3. **Оптимизация**: Использование одного базового меша для множества вариаций.
4. **Зашивка (Capping)**: Устранение "дыр" в меше после разрезания.

## 3. Логика разделения (Slicing) [DONE]
- [x] **Local-Space Slicing**: Срезание происходит в локальных координатах меша. Это исключает влияние поворота модели на результат резки.
- [x] **Async Processing**: Тяжелые расчеты вынесены в асинхронный TaskPool для предотвращения фризов UI.
- [x] **Infrastructure**:
    - [x] Глобальный ресурс `SlicingSettings`.
    - [x] Система классификации треугольников на основе центроидов.
- [x] **Triangle Splitting**:
    - [x] Реализовано расщепление треугольников, пересекающих плоскости (интерполяция UV/Normals).
    - [x] Генерация новых вершин.
    - [x] Создание 3-х финальных мешей: `Head`, `Body`, `Legs`.

## 4. Зашивка разрезов (Capping)
- [ ] **Edge Loop Extraction**: Сбор граничных ребер после разрезания.
- [ ] **Ear Clipping Triangulation**: Заполнение отверстий новыми полигонами.
- [ ] **Interior Materials**: Специальный материал для "внутренних" срезов.

## 5. Архитектура и Рефакторинг (SOLID) [DONE]
- [x] **Modular Systems**: Монолитный `systems_logic.rs` (900+ строк) разбит на 5 специализированных модулей: `input`, `normalization`, `slicing`, `sync`, `ui_logic`.
- [x] **Widget Components**: Файл `widgets.rs` (700+ строк) преобразован в папку с подмодулями: `panels`, `sliders`, `overlays`, `common`, `color_picker`.

    - [x] Прогресс-бар загрузки не скрывается до завершения первого цикла резки, обеспечивая бесшовный переход.
