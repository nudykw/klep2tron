# 04. Разрезание модели и зашивка

> [!IMPORTANT]
> Работа ведется исключительно в ветке **feat/actor-editor**.

## 1. Логика разделения (Slicing)
- [ ] Реализовать `SlicingSystem`, которая реагирует на изменение `ProjectResource::cut_top` и `cut_bottom`.
- [ ] Оптимизация: Пересчитывать меши только при отпускании слайдера (или с задержкой), чтобы избежать лагов.
- [ ] Алгоритм фильтрации вершин:
    - [ ] Группировка треугольников по их положению относительно плоскостей.
    - [ ] Если треугольник пересекается плоскостью — разделять его на два/три новых треугольника.

## 2. Зашивка (Capping)
- [ ] Реализовать алгоритм **Ear Clipping** (триангуляция многоугольника).
- [ ] Шаги алгоритма:
    1. Собрать все граничные ребра (edges) на плоскости разреза.
    2. Сформировать из них замкнутые контуры (loops).
    3. Выполнить триангуляцию каждого контура.
    4. Добавить новые грани в меш соответствующей части.

## 3. Дополнительные задачи
- [ ] **Unit-тесты**:
    - [ ] Тест на разрезание простого куба (8 вершин).
    - [ ] Тест на зашивку вогнутого контура (буква "L" или звезда).
