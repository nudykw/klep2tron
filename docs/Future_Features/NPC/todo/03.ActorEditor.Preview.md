# 03. Отображение модели в превью

> [!IMPORTANT]
> Работа ведется исключительно в ветке **feat/actor-editor**.

## 1. Камера
- [ ] Подключить `bevy_pan_orbit_camera`.
- [ ] Настроить ограничения (не давать камере уходить под "пол").
- [ ] Реализовать плавный зум и вращение.

## 2. Рендеринг и Окружение
- [ ] Настроить HDR-освещение или стандартный набор источников света (Key, Fill, Back).
- [ ] Добавить сетку (Grid) на "пол" для ориентации в пространстве.
- [ ] Реализовать отрисовку плоскостей разреза как полупрозрачных мешей.

## 3. Режим инспекции
- [ ] При активации инспекции части (Head/Body/Engine):
    - [ ] Скрывать сущности других частей.
    - [ ] Центрировать камеру на Bounding Box выбранной части.
    - [ ] Включать локальное вращение вокруг центра части.
