# 09. Горячие клавиши (Hotkeys)

> [!IMPORTANT]
> Работа ведется исключительно в ветке **feat/actor-editor**.

## 1. Базовое управление
- [x] **Ctrl + S**: Быстрое сохранение текущего состояния проекта.
- [ ] **Ctrl + Z / Ctrl + Y**: Undo / Redo операции (Ждет реализации системы Undo).
- [x] **Escape**: Выход в главное меню (с подтверждением).

## 2. Инструментарий
- [ ] **Delete**: Быстрое удаление выбранного сокета или элемента.
- [x] **Tab**: Быстрое переключение между режимами (Slicing / Sockets).
- [x] **G**: Переключение видимости сетки (Grid).
- [x] **K**: Переключение видимости сокетов.
- [x] **Z**: Переключение видимости гизмо манипуляции.
- [x] **L**: Блокировка/Разблокировка слайсера (Slicer Lock).
- [x] **+ / =**: Быстрое добавление нового сокета и переход в режим Sockets.
- [x] **I**: Циклическое переключение инспекции частей (Head -> Body -> Engine).
- [x] **R**: Сброс камеры вьюпорта.
