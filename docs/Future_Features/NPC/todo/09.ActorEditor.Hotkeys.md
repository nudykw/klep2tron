# 09. Горячие клавиши (Hotkeys)

> [!IMPORTANT]
> Работа ведется исключительно в ветке **feat/actor-editor**.

## 1. Базовое управление
- [ ] **Ctrl + S**: Быстрое сохранение текущего состояния проекта.
- [ ] **Ctrl + Z / Ctrl + Y**: Undo / Redo операции.

## 2. Инструментарий
- [ ] **Delete**: Быстрое удаление выбранного сокета или элемента.
- [x] **Tab**: Быстрое переключение между режимами (Slicing / Sockets).
- [ ] **G**: Переключение видимости вспомогательных Gizmos и плоскостей.
- [ ] **F**: Фокусировка камеры на выбранном элементе (сокет или часть модели).
