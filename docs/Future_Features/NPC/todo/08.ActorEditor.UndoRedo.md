# 08. Undo/Redo

> [!IMPORTANT]
> Работа ведется исключительно в ветке **feat/actor-editor**.

## 1. Система команд
- [x] Реализовать трейт `Command` с методами `execute` и `undo`.
- [x] Создать ресурс `ActionStack`, хранящий стек выполненных команд.
- [x] Реализовать автоматическую очистку стека Redo при совершении нового действия.

## 2. Реализация действий
- [x] Команды трансформации сокетов (перемещение, вращение).
- [x] Команда изменения цвета материала (с поддержкой Undo/Redo для колор-пикера).
- [x] Команда изменения параметров нарезки (с предотвращением циклических обновлений).
- [x] **[NEW]** Команды создания (`AddSocketCommand`) и удаления (`DeleteSocketCommand`) сокетов.

## 3. Интеграция и надежность
- [x] Добавить кнопки "Undo" / "Redo" в интерфейс редактора с визуальной индикацией доступности.
- [x] Ограничить максимальный размер стека истории (по умолчанию 50 действий).
- [x] **[FIX]** Исправлена паника при попытке Undo для удаленных сущностей (защита через `get_entity_mut`).
- [x] **[FIX]** Сокеты теперь привязаны к корню модели (`Actor3DRoot`), что позволяет им выживать после пересоздания мешей при нарезке.

## 4. Горячие клавиши
- `Ctrl + Z`: Undo
- `Ctrl + Shift + Z` или `Ctrl + Y`: Redo
