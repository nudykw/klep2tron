# 08. Undo/Redo

## 1. Система команд
- [ ] Реализовать трейт `Command` с методами `execute` и `undo`.
- [ ] Создать ресурс `ActionStack`, хранящий стек выполненных команд.

## 2. Реализация действий
- [ ] Команда изменения положения сокета.
- [ ] Команда изменения цвета материала.
- [ ] Команда изменения границ разреза.

## 3. Интеграция
- [ ] Добавить кнопки "Undo" / "Redo" в интерфейс редактора.
