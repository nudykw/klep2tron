# Задача 14.3: Система выделения (Lasso & Selection)

> [!IMPORTANT]
> Работа ведется исключительно в ветке **feat/actor-editor**.

> [!NOTE]
> Общее описание системы ручной нарезки находится в [Задаче 14](./14.Manual_Mesh_Slicing.md). В данной подзадаче необходимо реализовать только функционал, описанный ниже.
Реализовать инструменты выбора треугольников на меше в ручном режиме.

## Шаги
1.  **Lasso Tool:** Отрисовка произвольного экранного контура мышью.
2.  **Triangle Picking:** Алгоритм выбора треугольников, чьи вершины проецируются внутрь контура (только видимые треугольники).
3.  **Hierarchy Filter:** При захвате нескольких частей (например, Head + Body) оставлять в выделении только треугольники самой "высокой" части (Head > Body > Engine).
4.  **Brightness Highlight:** Визуализировать выделение через увеличение яркости (glow) треугольников.
5.  **UI Feedback:** Выводить количество выделенных треугольников в правую панель (цвет текста совпадает с цветом подсветки).
6.  **Управление:** Реализовать режимы `Add` (клик/драг) и `Subtract` (Alt + клик/драг), а также сброс (Esc / клик по пустому месту).

## Ожидаемый результат
Пользователь может выделять произвольные области на меше с помощью лассо, видя яркую подсветку и счетчик полигонов.
