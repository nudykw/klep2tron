# [x] 04. Система разрезания (Slicing)

## [x] 1. Логика разрезания
- [x] Реализовать алгоритм разделения меша на 3 части (Head, Body, Legs).
- [x] Добавление "крышек" (Caps) для закрытия отверстий в местах разреза.
- [x] **Rim Thickness:** Реализовать параметр толщины ободка для "пустых" моделей.
- [x] Асинхронный расчет (Async Compute) для тяжелых мешей.

## [x] 2. Управление (UI & Gizmos)
- [x] Визуальные плоскости разреза в 3D пространстве.
- [x] Интерактивные гизмо (Drag handles) для изменения высоты разреза.
- [x] Поля ввода высоты в инспекторе (Normalize 0.0 - 1.0).

## [x] 3. Оптимизация
- [x] Автоматический пересчет разрезов при изменении модели или её оптимизации.
- [x] Кеширование результатов для предотвращения мерцания при перемещении гизмо.
- [x] **Preview режим:** Быстрое вычисление контуров без создания мешей во время перетаскивания.
- [x] **Условное выполнение:** Полное разрезание только при подтверждении через механизм "кружочков".
- [x] **Асинхронное выполнение:** Использование AsyncComputeTaskPool для тяжелых операций.

## [x] 4. Оптимизация производительности (Preview Mode)

### [x] 4.1. Гибридный подход
- [x] Разделение на Preview и Full режимы разрезания
- [x] Быстрое вычисление контуров во время перетаскивания (5-10ms)
- [x] Полное разрезание только при подтверждении изменений
- [x] Визуальное различие между контурами:
  - **Черный** - старая гравировка во время перетаскивания
  - **Оранжевый** - новая preview гравировка во время перетаскивания
  - **Красный** - финальная гравировка после подтверждения

### [x] 4.2. Новые компоненты
- [x] `PreviewContours` - компонент для хранения preview контуров
- [x] `preview_contours_system` - система быстрого вычисления контуров
- [x] `contour_calculator` - модуль оптимизированных алгоритмов

### [x] 4.3. Результаты оптимизации
- [x] Ускорение в 10-20 раз во время перетаскивания
- [x] FPS увеличен с 4-14 до 83-166 при интерактивном использовании
- [x] Время на кадр снижено с 71-252ms до 6-12ms
