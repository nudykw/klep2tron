# Задача 14.1: Интерфейс и точная нарезка (Precision UI)

> [!IMPORTANT]
> Работа ведется исключительно в ветке **feat/actor-editor**.

> [!NOTE]
> Общее описание системы ручной нарезки находится в [Задаче 14](./14.Manual_Mesh_Slicing.md). В данной подзадаче необходимо реализовать только функционал, описанный ниже.
Улучшить панель Slicing и добавить возможность точного ввода параметров нарезки.

## Шаги
1.  **Правая панель (Slicing Panel):** Реализовать UI-блок `Auto Mode`.
2.  **Точный ввод:** Добавить текстовые поля для `Top Cut` и `Bottom Cut`.
3.  **Синхронизация:** Настроить двустороннюю связь:
    -   Движение 3D-гизмо во вьюпорте обновляет значения в текстовых полях.
    -   Ввод числа в текстовое поле перемещает соответствующее 3D-гизмо.
4.  **Rim Thickness:** Перенести контрол настройки толщины ободка из общего инспектора в блок `Auto Mode` панели Slicing.
5.  **Toggle:** Добавить кнопку переключения между режимами `Auto` и `Manual` (на данном этапе только визуальное переключение состояния ресурса).

## Ожидаемый результат
Пользователь может вводить точные значения нарезки в панели справа, видеть их синхронизацию с вьюпортом и управлять толщиной ободка в одном месте.
