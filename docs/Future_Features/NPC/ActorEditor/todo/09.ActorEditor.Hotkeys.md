# [x] 09. Горячие клавиши (Hotkeys)

## [x] 1. Управление вьюпортом
- [x] `F`: Центрирование на модели.
- [x] `W`: Переключение режима Wireframe.
- [x] `G`: Переключение видимости сетки (Grid).
- [x] `S`: Переключение видимости сокетов.

## [x] 2. Редактирование
- [x] `Ctrl + S`: Быстрое сохранение.
- [x] `Ctrl + Z`, `Ctrl + Y`: Undo/Redo.
- [x] `Delete`: Удаление выбранного сокета.
- [x] `Ctrl + D`: Дублирование сокета.

## [x] 3. Навигация по режимам
- [x] `1`: Режим Slicing.
- [x] `2`: Режим Sockets.
- [x] `3`: Режим Inspection.
