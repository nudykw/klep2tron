# Задача 14.5: Продвинутая интеграция (Optimization)

> [!IMPORTANT]
> Работа ведется исключительно в ветке **feat/actor-editor**.

> [!NOTE]
> Общее описание системы ручной нарезки находится в [Задаче 14](./14.Manual_Mesh_Slicing.md). В данной подзадаче необходимо реализовать только функционал, описанный ниже.
Обеспечить работу систем масштабирования и оптимизации в ручном режиме.

## Шаги
1.  **Per-part Scaling:** Изменить систему масштабирования так, чтобы в ручном режиме она применяла трансформацию к каждой из 3-х частей меша независимо.
2.  **Per-part Optimization:** Изменить систему оптимизации (decimation) так, чтобы в ручном режиме она оптимизировала каждую из 3-х частей меша как отдельный объект.
3.  **Safety Checks:**
    - Блокировка `Rim Thickness` в ручном режиме.
    - Предупреждение при переходе `Manual -> Auto`, если есть ручные правки.
4.  **UI Polish:** Финальная проверка тултипов и синхронизации всех элементов управления.

## Ожидаемый результат
Ручной режим полноценно интегрирован в пайплайн редактора. Модели с ручными правками можно оптимизировать и масштабировать.
